# LL3Mine2
Mine刷矿机2次适配版
## 安装流程

1.  安装BDS
2.  按照流程安装LiteLoader
3.  将压缩包内所有文件放入plugins

## 使用说明

1. 第一次开服自动创建配置文件(./plugins/LL3Mine2/config.json)
2. 请自主修改配置文件
3. 修改过程请使用物品标准名例：(minecraft:tnt)
4. 插件自动根据全部数据总和来自动计算刷出概率
5. 修改完成请重启服务器自动读取配置
